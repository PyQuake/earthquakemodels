Getting Started

Get the latest version

To get the last version you should clone the repository from gitHub to your local files. ```bash git clone https://github.com/PyQuake/earthquakemodels.git

How is the code organized

Most of the code were develop to be run in a python3 interpreter. You may find that some files can be bash executed.

The most important files are divided into three types of files: [1] the Genetic Algorithm (GA) files that create earthquake risk models, [2] the files that are the base to the GA files and [3] the files that connect them.

The files in [1] can be found, mostly, at ./code/gaModel. The files in [2] can be found, mostly, at ./code/csep or ./code/earthquake The files in [3] can be found, mostly, at ./code/models

It is possible to use the code that I used in most of my experiments. They are located at ./code/runExperiments
